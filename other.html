Can I download faster?

Yes, you can!

软件安装包 Linux源镜像 Docker镜像 其他源镜像 临时传文件 View on GitHub

很多时候我们想要下载软件，但是苦于几 kb 的网速，本仓库就是来解决这个问题的，本仓库将会收集一些软件安装包，或者其他静态资源的下载地址。

其中有些链接的状态可能更新不及时，如有发现，请提 issue 或 pull request。

其他镜像源

Gradle

https://maven.aliyun.com/mvn/guide

build.gradle
repositories {
    maven {
        url 'https://maven.aliyun.com/repository/public/'
    }
    mavenCentral()
}
build.gradle.kts
repositories {
    maven {
        url = uri("https://maven.aliyun.com/repository/public")
    }
    mavenCentral()
}

Maven

https://maven.aliyun.com/mvn/guide 

<mirror>
    <id>aliyunmaven</id>
    <mirrorOf>*</mirrorOf>
    <name>阿里云公共仓库</name>
    <url>https://maven.aliyun.com/repository/public</url>
</mirror>

Ruby Gems

  • 查找默认源

gem sources

  • 移除默认源

gem sources --remove https://rubygems.org/

  • 添加新源

https://developer.aliyun.com/mirror/rubygems 

gem sources -a https://mirrors.aliyun.com/rubygems/
# 如果你使用 Gemfile 和 Bundler
bundle config mirror.https://rubygems.org https://mirrors.aliyun.com/rubygems
